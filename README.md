# 🌐 Agentathon 2026 - Official Event Landing Page

The official marketing and informational landing page for **Agentathon 2026** held at BMSIT&M. Built using high-performance static HTML/CSS with modular panels.

## ✨ Features
- **Event Timeline**: Responsive schedule and countdown timers for the hackathon stages.
- **Guest Speakers & Mentors**: Dynamic profiles of industry leaders guiding the event.
- **Registration Linkouts**: Direct integration with registration portals and information desks.
- **Beautiful Glassmorphism Theme**: Uses radial gradients, glass effects, and clean typography.

## 🛠️ Tech Stack
- HTML5, CSS3 (Vanilla Custom Layouts)
- Google Fonts (Poppins, Montserrat, Bebas Neue)
